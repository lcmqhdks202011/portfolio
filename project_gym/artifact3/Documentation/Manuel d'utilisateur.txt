Pour créer un compte professionnel ou membre, il faut choisir l’option « Membre » puis choisir le type de compte que l’on veut créer et ajouter les informations requises dans les champs ‘Name’, ‘First Name’, ‘Adress’ et ‘Telephone’ puis appuyer sur le bouton ‘Add’. Pour modifier un compte, il suffit de sélectionner le type de compte et remplir les champs de la même façon que pour créer un compte, mais avec en plus le numéro de compte à modifier et cliquer sur ‘Update’. Pour effacer un compte il faut simplement cliquer sur le bouton ‘Delete’ après avoir entré le numéro du compte à effacer. Pour retourner sur le menu principal il suffit toujours d’appuyer sur le bouton ‘Home’.
Dans l’interface ‘Repertoire Service’ on peut créer des services en cliquant sur le bouton ‘+ Add new’ puis en sélectionnant le jour de la semaine désiré et en remplissant les cases avec les informations nécessaires, puis en appuyant sur le bouton ‘Add’. Si l’on veut abandonner la création de service il suffit d’appuyer sur le bouton ‘Cancel’. Pour modifier un service il suffit de le sélectionner dans la barre en dessous et de cliquer sur ‘Modifier’, ensuite il faudra modifier les informations comme souhaité. On peut toujours utiliser la barre de recherche qui nous montre tous les services qui correspondent aux critères écrits. Pour inscrire un client il suffit de sélectionner le service désiré, puis d’appuyer sur le bouton ‘Inscrire’ et d’entrer le code du client. Pour supprimer un service il suffit de le sélectionner puis d’appuyer sur le bouton ‘Supprimer’ et de confirmer. Pour la désinscription à des activités, il suffit de sélectionner le cours voulu et d’entrer le numéro du client souhaitant se désinscrire.
Pour vérifier si un client à accès aux services il faut entrer leur code client dans la boîte de texte présente lorsqu’on clique sur ‘Confirmer’ sur la page d’accueil. Par la suite, l’employé pourra laisser passer le client s’il a accès aux services.
Pour l’impression du rapport, il faut, sur la page principale appuyer sur le bouton ‘Rapport’, le fichier sera créé dans le dossier du programme. Ce rapport sera fait automatiquement les vendredis et comprendra une liste de tous les professionnels ayant donné des cours et leurs frais, ainsi que le nombre total de professionnels leurs cours et les frais totaux pour cette semaine là.
À tous les vendredis, un enregistrement contenant toutes informations des professionnels et des montants qui leurs sont du sera envoyé à la banque et elle gèrera les paiements. Tous les membres recevront aussi une liste de tous les services qui leur auront été fournis lors de la semaine. Le système enverra aussi à chaque professionnel un rapport de tous les services qu’ils ont fourni au courant de la semaine et des montants qu’ils devraient être payés.
